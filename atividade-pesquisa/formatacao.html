Explicação sobre o elemento HTML Itálico.

O itálico serve para deixar as letras inclinadas a direita, pode ser usado para destacar uma frase ou palavra, para utiliza-so em seu código HTML se usa o elemento; "<i>" fazendo que qualquer texto dentro dele fique em Itálico.
